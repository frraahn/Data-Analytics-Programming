
# 📊 Data Analytics Programming

Welcome to my repository for the Data Analytics Programming course at Universiti Teknologi Malaysia (UTM). This space serves as an academic portfolio documenting our end-to-end data processing workflows, analytical assignments, and collaborative data science projects.

---

## 📂 Repository Structure & Portfolio Breakdown

### 🔹 Group Project: Vehicle Fuel Consumption & CO₂ Emissions Analysis
* **General Summary:** Designed and built an end-to-end analytical pipeline to study the relationship between vehicle engine sizes, fuel consumption metrics, and carbon emissions. Phase 1 focused on data cleaning, structuring, and multi-variable aggregation using Pandas. In subsequent phases, we performed Exploratory Data Analysis (EDA) and implemented advanced clustering and statistical modeling to group high-, medium-, and low-emitting automotive brands, providing clear data-driven insights targeted at carbon reduction strategies.
* **Core Deliverables:**
  * Cleaned & Summarized Dataset Engineering (`aggregated_fuel_data.csv`)
  * Feature Correlation Matrix & Multivariable Cluster Visualizations
  * Complete Jupyter Analytical Pipeline Codebase (`DAP Latest.ipynb`)

### 🔹 Assignments & Practical Lab Assessments: Data Engineering & Statistical Analytics
* **General Summary:** A comprehensive collection of structural programming implementations and data handling assessments using Python. The coursework covers fundamental file management systems, automated calculations using core collections, advanced descriptive statistical aggregation across complex indexes, and visual storytelling via customized multi-axis charting libraries.
* **Core Deliverables:**
  * **Assignment 1:** List, Tuple, Dictionary, Custom Functions, and Driver-Ride Data File I/O Streams (`A1_DayangFarraSafiya.ipynb`).
  * **Assignment 2:** Advanced Retail Sales Data Manipulation, MultiIndexing Filters, and Multi-Category Aggregations using Pandas DataFrames (`Assignment 2_DayangFarraSafiya.ipynb`).
  * **Assignment 3:** Weather Station Rainfall Data Analysis, Data Filtering, Annotations, and Facet Grid Spatial Trend Visualizations using Seaborn (`A3_SAFIYANURSYAHADAH.ipynb`).

## 💡 Course Reflection & Key Takeaways

This course provided me with a clear roadmap for transforming messy, unstructured real-world data into clean, actionable business intelligence. Working on the automotive emission analysis project taught me how to systematically approach data exploration—using custom Python functions for initial data preparation, moving into advanced multi-index filtering in Pandas for structured aggregations, and utilizing professional plotting libraries like Seaborn to isolate environmental variables.
